# OCR my Aadhaar
Using TesseractOcr and OpenCv this project can sucessfull extract adharnumber, Dateofbirth and gender from any version of aadhaar 

## Features

- Extract data from adharcard 
1) Date of Birth
2) Aadhar Number
3) Gender
